# V-Tutor Client

React + TypeScript + Vite frontend for the V-Tutor exam management platform.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

The client will start at http://localhost:3000

## Pages

- `/giao_vien` - Teacher login
- `/giao_vien/{teacher_id}` - Teacher dashboard (create exams, view results)
- `/hoc_sinh/{exam_id}` - Student exam page
